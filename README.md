
# App Builder API
   
   Automated app generation using LLM and GitHub API.
   
   ## Features
   - Generates web apps from text descriptions
   - Creates GitHub repositories automatically
   - Deploys to GitHub Pages
   - Uses Groq AI for code generation
   
   ## Tech Stack
   - Node.js + Express
   - Groq AI API
   - GitHub API
   - Deployed on Render
